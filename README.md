### iCityAction

iCityAction是基于React-Native实现的新闻类APP，接口数据来自《都市频道》，持续更新中。
目前只针对iOS端，完成的功能（只添加下拉刷新，暂未添加上拉加载更多功能，仍有许多隐藏bug，努力修复中）：
>* 首页展示、新闻详情、评论列表
>* 搜索、搜索结果
>* 视频列表
